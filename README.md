# Buraq Minds Website

Premium Next.js website for Buraq Minds, an AI-first software, cybersecurity, and product engineering company.

## Stack

- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion
- Firebase
- Claude API integrations

## Brand

- Tagline: Intelligence. Security. Innovation.
- Primary red: `#c0392b`
- Dark background: `#0d0d0d`
- Heading font: Cormorant Garamond
- Body font: Outfit

## Development

```bash
npm run dev
npm run build
```

Do not modify DNS, hosting, email, or Firebase secret configuration without explicit approval.
